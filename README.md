This project include player controller, y-sorting and all features from Base-Project

How the project should continue ?

In tiled Every tileset and objects should have a "OFFSET_Y" variable for y-sorting, if you want to interact or y-sorting
with an object you should put that with object layer. And other tilesets grass or paths need to be tileset layer.
If an object gonna have a collision you need to set and hitbox in tileset settings in tiled. You can configuer shapes with
ready functions.
